CSS Animations

Օրինակներ

CSS-ում կարելի է ստեղծել բավականին բարդ անիմացիաներ CSS animations-ով առանց JavaScript օգտագործման: Վառ CSS animations-ի օրինակներ են

  • loader – ները (բեռնիչներ)՝ տարրեր, որոնք ցույց են տալիս, որ ինչ-որ բան բեռնվում է էջում;
  • hover – ները, որոնք ակտիվանում են, երբ մկնիկի ցուցիչը (cursor) դիպչում է տարրին;
  • active – ները, որոնք ակտիվացնում են տարրը, երբ մկնիկի կուրսորը սեղմվում է (mouse down) նրա վրա:
