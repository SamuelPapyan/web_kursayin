CSS-անիմացիան էկրանի վրա տարրերը շարժելու ամենապարզ միջոցն է:

Օգտագործելով CSS-ը, կարելի է ստեղծել կադր առ կադր անիմացիաներ, երբ նկարը, որի վրա տարբեր վիճակներում գծված է առարկան, վերածվում է շարժման:

CSS-անիմացիան էջի օբյեկտները կենդանացնելու ամենահեշտ ձևն է: Այն թույլ է տալիս պտտել, ձգվել, փոքրացնել և մեծացնել օբյեկտը, տեղափոխել այն կամ կատարել այլ գործողություններ:

CSS-անիմացիաները բաղկացած են 2 բաղադրիչից՝

  • անիմացիայի ոճի նկարագրություն և
  • հիմնական կադրերի մի շարք, որոնք սահմանում են անիմացիոն ոճերի սկիզբը, ավարտը և, հնարավոր է, միջակայքում գտնվող վիճակները:

CSS-անիմացիաները Cascading Style Sheets – ի համար առաջարկվող մոդուլ է, որը թույլ է տալիս HTML փաստաթղթի տարրերի անիմացիա՝ օգտագործելով CSS:

CSS-անիմացիայի համար անհրաժեշտ է 2 տարր.

  • Օբյեկտ, որը պետք է անիմացիոն լինի: Այն գրվում է HTML ֆայլում։
  • Օբյեկտի և անիմացիայի պարամետրեր: Դրանք գրվում են CSS ֆայլում:

ՀԻՇԵՔ, ՈՐ ԱՆԻՄԱՑԻԱՆ ԱՇԽԱՏԻ, CSS ՀՂՈՒՄԸ ՊԵՏՔ Է ԳՐՎԻ HTML ՓԱՍՏԱԹՂԹՈՒՄ

CSS-անիմացիայի 3 առավելությունները ավանդական մեթոդների նկատմամբ.

  2. CSS-անիմացիաները լավ են աշխատում նույնիսկ համակարգի չափավոր բեռնվածության դեպքում: Պարզ JavaScript անիմացիաները, եթե վատ են գրված, հաճախ վատ են աշխատում:
  3. CSS-անիմացիան թույլ է տալիս դիտարկիչին կառավարել անիմացիայի հաջորդականությունը՝ դրանով իսկ օպտիմալացնելով դիտարկիչի աշխատանքը և արդյունավետությունը: Օրինակ՝ նվազեցնել անիմացիոն կադրերի թարմացման արագությունը ներդիրներում, որոնք ներկայումս չեն դիտվում:
