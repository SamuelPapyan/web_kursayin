Keyframes

WEB անիմացիան ենթադրուն է օբյեկտի սահուն անցում մի վիճակից մյուսը: Անիմացիան սահմանվում է առանցքային կադրերով, որոնք նշում են տարրի տեսքը և դիրքը որոշակի կետերում: CSS-ի անցումներն օգտագործում են երկու հիմնական կադրեր (սկիզբ և ավարտի արժեք): CSS անիմացիաները թույլ են տալիս կարգավորել անցումը @keyframes կանոնի միջոցով:

Keyframes-ները «առանցքային կադրեր» են, որոնք օգտագործվում են վեբ էջերի անիմացիաներ ստեղծելու համար: Նրանք ցույց են տալիս, թե ինչ տեսք կունենա անիմացիոն տարրը ժամանակի որոշակի կետում: CSS ոճավորման լեզվում @keyframes-ը կանոն է, որը նկարագրում է «առանցքային կադրի» ստեղծումը։

@keyframes կանոնը թույլ է տալիս ստեղծել անիմացիա՝ օգտագործելով առանցքային կադրերի մի շարք, այսինքն՝ այն նկարագրում է օբյեկտի վիճակը ժամանակի որոշակի կետում։

Եթե ​​հարկավոր է ցույց տալ, թե ինչպես է պատկերը մեծանում չափերով, այն պետք է ունենա առնվազն 2 հիմնական կադր՝ սկզբնական և վերջնական: Այս վիճակների միջև միջանկյալ տարբերակները կկազմի WEB-զննիչը: Եթե ​​փոփոխությունը անհավասար է, կարող են պահանջվել միջանկյալ առանցքային կադրեր:

2 առանցքային կադրերի ամենապարզ CSS կանոնի տեսքը ՝

  • Keyframe-ի անունը եզակի նույնացուցիչ փոփոխական է: Երբ հիմնական կադրերը ստեղծված են, պետք է keyframe-ը փոխանցել ցանկալի տարրի ոճի նկարագրությանը: Այն փոխանցվում է անունով. այսպես CSS-ը «գիտի», որ որոշակի նույնացուցիչով կանոնը պետք է կցվի տարրին:
  • From և to բառերը թարգմանվում են որպես «ից» և «մինչև» և նշանակում են անիմացիայի մեկնարկային և ավարտական ​​կետերը: From-ից հետո ձևավոր փակագծերում գրվում է տարրի նախնական վիճակը՝ դիրքը, չափը, գույնը և այլ պարամետրեր: to-ից հետո ձևավոր փակագծերում գրվում է տարրի համապատասխանաբար վերջնական վիճակը: Երբ անիմացիան սկսվի, զննարկիչը կամաց-կամաց կփոխի տարրի պարամետրերը՝ այն նախնական վիճակից վերջնական վիճակի «բերելու» համար:
  • Տոկոս. From և to-ի փոխարեն կարելի է օգտագործել 0% և 100%, իսկ եթե պահանջվում են միջանկյալ տարբերակներ՝ այլ տոկոսային արժեքներ: 0% -ից մինչև 100% արժեքը ցույց է տալիս, թե անիմացիայի որ կետում տարրը պետք է ընդունի նկարագրված պարամետրերը: Օրինակ, եթե գրվում է.
    33%՝ {լայնություն՝ 100px }
    ապա անիմացիայի երկարության 33%-ից հետո տարրի լայնությունը կկազմի 100px: Տոկոսները կարող են օգտագործվել բարդ անիմացիաներ ստեղծելու համար:

Որպեսզի անիմացիան աշխատի, @keyframes կանոնը պետք է միացված լինի որոշակի տարրի կամ տարրերի խմբի ոճի նկարագրությանը: Այս նկարագրությունը կոչվում է selector (ընտրիչ)՝ CSS կոդի բլոկ, որտեղ նշվում են սահմանված բնութագրերով տարրերի պարամերրերը:

Անիմացիայի նկարագրությունը պետք է ավելացնել selector-ի մեջ, օգտագործելով բարդ animation հատկությունը կամ առանձին ավելի փոքր հատկությունները՝ animation-name, animation-duration և այլն:

Առանցքային կադրի անունը փոխանցվում է animation կամ animation-name հատկությանը: Այնուհետև անիմացիայի տևողությունը նշվում է վայրկյաններով կամ միլիվայրկյաններով: Անհրաժեշտության դեպքում կարելի է ավելացնել անիմացիայի հետաձգում, կրկնությունների քանակ, փոփոխություններ յուրաքանչյուր նոր կրկնության հետ և այլ տարբերակներ:

@keyframes

Հատկանշում է անիմացիա` օգտագործվում է առանցքային կադրերի ստեղծելու համար։

            @keyframes mymove {
                from {top: 0px;}
                to {top: 200px;}
            }
            /* Chrome, Safari, Opera */ 
            @-webkit-keyframes mymove {
                from {top: 0px;}
                to {top: 200px;}
            }
