animation-duration

Հատկանշում է, թե քանի վայրկյան կամ միլիվայրկյան է անհրաժեշտ անիմացիային մեկ ցիկլ ավարտելու համար:

div {
    -webkit-animation-duration: 2s; /* Chrome, Safari, Opera */
    animation-duration: 2s;
}
Արժեք Բնութագիր
time Հատկանշում է տևողությունը, որը հարկավոր է անիմացիայի ավարտի համար: Նախնական արժեք՝ 0, նշանակում է անիմացիա չի լինելու:
initial Վերագրում է այս հատկությունն իր նախնական արժեքին:
inherit Ժառանգում է այս հատկությունն իր ծնող տարրից:
