animation-iteration-count

Հատկանշում է, թե քանի անգամ անիմացիան պետք է խաղա:

div {
    -webkit-animation-iteration-count: 3; /* Chrome, Safari, Opera */
    animation-iteration-count: 3;
}
Արժեք Բնութագիր
number Թիվ, որը սահմանում է, թե քանի անգամ անիմացիան պետք է խաղա: Նախնական արժեք՝ 1:
infinite Հատկանշում է, որ անիմացիան պետք է խաղա անսահմանափակ քանակությամբ:
initial Վերագրում է այս հատկությունն իր նախնական արժեքին:
inherit Ժառանգում է այս հատկությունն իր ծնող տարրից:
