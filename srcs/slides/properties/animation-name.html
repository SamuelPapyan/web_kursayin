animation-name

Հատկանշում է անվանում @keyframes անիմացիայի համար:

div {
    -webkit-animation-name: mymove; /* Chrome, Safari, Opera */
    animation-name: mymove;
}
Արժեք Բնութագիր
keyframename Հատկանշում է keyframe-ի անվանումը, որը պետք է կապված լինի սելեքթորի հետ:
none Հատկանշում է, որ անիմացիա չի լինելու (կարող է օգտագործվել կասկադից եկող անիմացիաները փոփոխելու համար):
initial Վերագրում է այս հատկությունն իր նախնական արժեքին:
inherit Ժառանգում է այս հատկությունն իր ծնող տարրից:
