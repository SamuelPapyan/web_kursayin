animation-play-state

Հատկանշում է, արդյո՞ք անիմացիան պետք է սկսի, թե կանգնեցված լինի:

div {
    -webkit-animation-play-state: paused; /* Chrome, Safari, Opera */
    animation-play-state: paused;
}
Արժեք Բնութագիր
paused Հատկանշում է, որ անիմացիան կանգնեցված է:
running Հատկանշում է, որ անիմացիան խաղում է:
initial Վերագրում է այս հատկությունն իր նախնական արժեքին:
inherit Ժառանգում է այս հատկությունն իր ծնող տարրից:
