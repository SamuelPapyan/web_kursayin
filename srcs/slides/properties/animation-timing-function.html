animation-timing-function

Հատկանշում է անիմացիայի կորության արագությունը:

div {
    -webkit-animation-timing-function: linear; /* Chrome, Safari, Opera */
    animation-timing-function: linear;
}
Արժեք Բնութագիր
cubic-bezier(n,n,n,n) Թույլատրելի արժեքներն են 0-ից 1 թվային արժեքները:
ease Անիմացիան կսկսի դանդաղ, այնուհետև կարագանա մինչ դանդաղ ավարտվելը:
ease-in Անիմացիան կունենա դանդաղ սկիզբ:
ease-in-out Անիմացիան կունենա դանդաղ սկիզբ և դանդաղ ավարտ:
ease-out Անիմացիան կունենա դանդաղ ավարտ:
linear Անիմացիան սկզբից մինչև վերջ կունենա նույն արագությունը:
initial Վերագրում է այս հատկությունն իր նախնական արժեքին:
inherit Ժառանգում է այս հատկությունն իր ծնող տարրից:
