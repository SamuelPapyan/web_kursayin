CSS Animations

Ներածություն

Անիմացիան ժամանակակից վեբ ռեսուրսների կարևոր բաղադրիչն է: Հաճախ այն հանդիսանում է կայքի հետ օգտատիրոջ փոխազդեցության կապող հանգույց: Օգտագործելով անիմացիոն օբյեկտները, կարելի է վերահսկել օգտատերերի վարքագիծը և առաջնորդել նրանց որոշումներ կայացնելիս: Համապատասխան, գրագետ անիմացիան ոչ միայն հաճելի է աչքի համար, այլ և նպաստում է կոնվերսիայի (փոխակերպման) աճին:

WEB անիմացիան լինու է 3 տեսակ

  • CSS-անիմացիա
  • SVG-անիմացիա
  • JavaScript-անիմացիա

Ցանկացած անիմացիա - դա տարրի մի վիճակից մյուս վիճակի անցում է: Տարիներ շարունակ տարրական անիմացիաներ ստեղծելու համար օգտագործվել է :hover կեղծ-դասը։ CSS-ի ընդլայնումները դեպի անիմացիայի ոլորտ նվազագույն են եղել մինչև 2007թ-ին, երբ WebKit-ը հայտարարեց իր մտադրության մասին՝ ներառելու CSS անիմացիան, անցումները և փոխակերպումները որպես WebKit-ի առանձնահատկություններ: Իսկ 2009թ.-ի փետրվարին այն նաև հայտարարեց CSS-ի միջոցով ինչպես անուղղակի, այնպես էլ բացահայտ անիմացիայի իրականացման մասին:

2011 թվականի հունիսի դրությամբ Firefox 5-ը ներառում է CSS անիմացիաների աջակցություն։ CSS անիմացիան հասանելի է նաև որպես մոդուլ WebKit-ի, ինչպես նաև Google Chrome-ի, Safari 4-ի և 5-ի և Safari-ի համար iOS-ի համար (iPhone, iPod Touch, iPad), Android 2.x և 3.x տարբերակների, Internet Explorer 10-ի գիշերային նախագծերում: + և Microsoft Edge դիտարկիչը՝ BlackBerry OS 6 վեբ բրաուզերը, -webkit- նախածանցով։ Այն նաև օգտագործվում է iTunes 9-ում՝ iTunes LP ֆայլերը աջակցելու համար:
